# Tutorial Website

Isang simpleng **tutorial-style na website** na naggagabay sa mga user sa mga pangunahing hakbang ng pagbuo ng static web page. Mula sa pagpapakilala ng HTML hanggang sa pag-style gamit ang CSS, ipinapakita nito ang step-by-step na proseso ng paglikha ng isang basic na web page.

## Tampok
- Step-by-step guide sa HTML at CSS
- Simpleng layout para sa madaling pagsunod
- Ideyal para sa mga baguhan sa web development
